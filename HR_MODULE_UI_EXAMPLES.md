## Kleurenschema

### In de UI:
- **Positief saldo:** `text-green-600` (#059669)
- **Negatief saldo:** `text-red-600` (#DC2626)
- **Aangevraagd:** `text-blue-600` (#2563EB)
- **Neutraal:** `text-slate-900` (#0F172A)
- **Waarschuwing achtergrond:** `bg-amber-50` (#FFFBEB)
- **Waarschuwing border:** `border-amber-200` (#FDE68A)
- **Info achtergrond:** `bg-blue-50` (#EFF6FF)
- **Saldo paneel:** `bg-white` + `border-slate-200`

## Responsive Gedrag

### Desktop (> 768px):
- Volledige saldo berekening paneel
- Alle details zichtbaar
- Side-by-side layout waar mogelijk

### Tablet (768px - 1024px):
- Iets compacter paneel
- Alle informatie nog steeds zichtbaar
- Gestackte layout

### Mobile (< 768px):
- Geoptimaliseerd compact paneel
- Gestackte layout
- Kleinere font sizes maar nog steeds leesbaar
- Touch-friendly buttons
